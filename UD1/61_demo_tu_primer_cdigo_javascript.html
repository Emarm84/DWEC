6.1.- Demo: Tu primer código JavaScript

Código fuente

Archivo index.html

<!DOCTYPE html>
<html>
	<head>
		<title>Demo: tu primer código JavaScript</title>
		<meta name="viewport" content="width=device-width, initial-scale=1">
	</head>
	<body>
		<h1>Demo: tu primer código JavaScript </h1>
		<p>En este primer ejemplo puedes ver cómo incluir un script en nuestra web.</p>
		<p>Pulsa el botón derecho de tu ratón en esta web y "Ver código fuente de esta página", ahora fíjate cómo la etiqueta "script" contiene el código JavaScript.</p>
		<p>Esta primera web muestra el mensaje "Prueba de JavaScript" en un cuadro de alerta cuando carga la página.</p>
		<p>Intenta analizar el orden de ejecución del código.</p>
		<p>Observa cómo muestra el mensaje antes de "pintar" la página y cómo cuando le das a "aceptar" continúa renderizando el html</p>
		<p>Pulsa F5 para recargar la página y comprobar otra vez el orden de ejecución</p>
		<script>
			alert("Prueba de JavaScript");
		</script>
		<h2>Después de JavaScript</h2>
		<p id="demo"></p>
		
	</body>
</html>
